# kahoot-gefickt

Kahoot flood bot with server-side connections.

## Deploy

1. Push to GitHub
2. Connect to [Render.com](https://render.com) → New Web Service
3. Set build command: `npm install`
4. Set start command: `node server.js`

## Local

```bash
npm install
node server.js
# open http://localhost:3000
```
